TextRecognition
---------------------
Приложение с помощью OpenCV и Tesseract распознает текст на экране телефона.
1. Для установки приложения установите TextRecognition.apk и скопируйте папку tesseract на SD-карту телефона.
2. Папка tesseract содержит веса для русского и английского языков. 
3. Для переключения между языками переименуйте внутреннюю папку соответствующего языка в tessdata.
4. По умолчанию приложение использует русский язык и ищет веса в каталоге tessdata.
5. Запустите приложение и сфотографируйте текст, распознавание начнется автоматически. Время распознавания зависит от количества текста на снимке документа и скорости работы процессора телефона, приложение работает оффлайн.

Приложение успешно распознает текст при хорошем качестве изображения, но при съемке с мобильного телефона фокусное расстояние редко составляет менее 40 см, поэтому качество работы приложения снижается.

Приложение разработано и протестировано под Android 4.4
Требуется разрешение доступа к камере.
